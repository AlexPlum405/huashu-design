# Inspiration Sources

这些来源用于 Reference Intake。它们是审美校准和模式提取来源，不是素材库。只采公开可访问内容，不绕登录、不抓付费内容、不批量下载图片。

## Source Categories

| Use case | Default sources | Best for | Notes |
|---|---|---|---|
| Web / landing / product sites | Awwwards, Behance, Dribbble | 视觉方向、排版气质、首屏构图、品牌表达 | Awwwards 适合前沿网页；Behance 适合完整项目过程；Dribbble 适合局部 UI 细节 |
| Mobile App UI | Mobbin, UI Pocket | 页面类型、信息架构、移动端模式 | Mobbin 可能需要登录/订阅；UI Pocket 公开程度通常更高 |
| Interaction / flows / motion | Pageflows, Design Spells | 真实用户流程、微交互、动效进入/退出 | Pageflows 可能只公开摘要；Design Spells 适合交互动效细节 |
| Color palettes | Coolors, Color Hunt | 初始配色探索、替代色板 | 只能辅助，品牌项目优先 `brand-spec.json` |
| UI components | uiverse | 组件实现灵感、小型交互片段 | 只当实现参考，不当审美方向 |
| Fonts | Google Fonts | 可用字体组合、中文/英文 fallback | 字体选择必须服务品牌气质和阅读场景 |

## Route Defaults

| Route | First attempt | Second attempt | Fallback |
|---|---|---|---|
| App prototype | Mobbin, UI Pocket | Pageflows | 用户截图 / existing product context / skip |
| Web prototype | Awwwards, Behance | Dribbble | 品牌官网 / competitor URL / skip |
| Desktop app | Mobbin if relevant, Pageflows | Behance | 用户截图 / product docs / skip |
| Slides / infographic | Behance | Awwwards for visual tone | Existing decks / scene templates / skip |
| Motion / video | Pageflows, Design Spells | Awwwards motion-heavy sites | Existing motion demos / skip |
| Tweaks / components | uiverse, Dribbble | Design Spells | Existing component library / skip |
| Critique | User-provided artifact first | Closest route above | No external reference needed |

## Extraction Targets

提炼这些，不复制具体作品：

- Layout: 首屏结构、列表/详情/设置页组织、网格比例。
- Hierarchy: 标题、CTA、辅助信息、状态反馈的优先级。
- Density: 每屏信息量、卡片密度、留白比例。
- Interaction: 进入/退出、hover/tap、加载、错误和成功状态。
- Motion: 节奏、持续时间、缓动、关键帧角色。
- Tone: 冷静、前卫、温暖、专业、娱乐等视觉温度。

## Do Not Copy

- 不复制截图、插图、图标、品牌资产。
- 不复制文案、数据、案例名。
- 不把参考站的风格直接当用户品牌风格。
- 不用付费/登录内容作为不可验证依据。
