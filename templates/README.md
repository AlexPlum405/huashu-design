# Huashu-Design Templates

These templates are starting points for the Junior Designer pass. They are not
final designs and must not be shipped without the full workflow:

1. Complete task routing and required questions.
2. Collect design context / brand assets.
3. Write assumptions and placeholders.
4. Get user confirmation at checkpoints.
5. Fill the real design, add variations/Tweaks.
6. Run `scripts/design_lint.py` and `scripts/verify.py`.

Use the smallest matching template:

| Template | Use for |
|---|---|
| `landing-page.html` | Website / landing page / web mockup |
| `app-overview.html` | Multi-screen mobile overview review |
| `app-flow.html` | Clickable flow logic before final iOS/Android frame |
| `deck-showcase.html` | 2-slide grammar preview before a full deck |
| `motion-stage.html` | Motion grammar preview before the full animation engine |
| `infographic.html` | One-page visual explainer / data poster |
